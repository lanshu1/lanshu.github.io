新建koa2项目

1.npm install -g koa-generator

2.koa2 项目名称，如果需要ejs引擎koa2 -e 项目名称

3.cd 项目名称

4.npm install

5.npm install –update-binary

6.npm run dev

小知识点：对于start和test可以使用npm，对于dev和prd必须使用npm run来启动
